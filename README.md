# Graph2GNNs
An Empirical Study Of Neural Network Based Graph Representations For Software Defect Prediction
### Repository Structure
- `GNN` : The datasets of all experiments.

- `skesd_result_CPDP` : Figures of the first-round and second-round SKESD test results on AUC, F1, and MCC metrics in the CPDP task.

- `skesd_result_WPDP` : Figures of the first-round and second-round SKESD test results on AUC, F1, and MCC metrics in the WPDP task.

- `utilities` : The code utilities used.

- `CSVGraphDataset.py` : The python script for converting the dataset into PyG-compatible input.

- `Graph2GNNs_demo.py` : Python script for the experiments.
